### 科目カラム (K列以降, 68科目分)
各科目名がそのままカラム名になっている。値のパターン：
- `○1` / `○2` / `○3` = 履修中（数字は学年）
- `○1N` = 履修中（Nの意味は要調査）
- `●` / `●1` / `●2` / `●3` = 修得済み（数字は修得した学年）
- 空欄 = 未登録

**特徴:**
- 横持ちテーブル（科目をカラムとして展開）
- 科目追加時はカラム追加が必要 → スキーマ変更が頻繁に発生するリスク
- 78列という幅広なテーブル
